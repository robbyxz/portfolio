CASE STUDY: SPAM EMAIL DETECTION & CLASSIFICATION

I. Business Problem

Developed an automated machine learning pipeline to classify email and SMS messages as Ham or Spam, reducing phishing exposure while minimizing false positives for legitimate communications.

II. Technical Approach

  • Data Preparation: Text cleaning, tokenization and feature engineering.
  • Model Comparison: Evaluated Multinomial Naive Bayes, Bernoulli Naive Bayes and a Recurrent Neural Network (RNN).
  • Evaluation: Compared models using Precision, Recall, F1-score and ROC-AUC to identify the most suitable business solution.

III. Business Value

Demonstrated how different machine learning models support different operational priorities, enabling scalable spam detection while protecting legitimate customer communications.

CASE STUDY: FRAUD DETECTION SYSTEM

I. Business Problem

Built a machine learning solution to identify fraudulent credit card transactions, helping financial institutions reduce fraud losses while minimizing disruption to legitimate customers.

II. Technical Approach

  • Data Preparation: Data cleaning, feature engineering and categorical encoding.
  • Class Imbalance: Applied SMOTE to address the low fraud rate (0.58%).
  • Model Comparison: Benchmarked Logistic Regression, Random Forest and XGBoost using business-focused evaluation metrics.

III. Business Value

Demonstrated how predictive analytics can support real-time fraud detection, reducing financial losses while improving operational efficiency and customer trust.
